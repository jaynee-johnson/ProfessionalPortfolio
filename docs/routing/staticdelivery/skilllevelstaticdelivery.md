# Skill-Level Static Delivery

**Note**: This page is specific to a static delivery routing strategy. If your organization uses a dynamic delivery routing strategy, you can [find this information](../dynamicdelivery/skillleveldynamicdelivery.htm) in the dynamic delivery online help.

In the skill profile, you can specify the priority of an interaction as it enters the queue through that skill. That priority increases over time, at a rate that you configure. This gives you control over which types of interactions route first.
