24.2 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Pipeline Update 159

Published 9 July 2024*

ACD

  • When contacts occurred at the same time as an AWS credential refresh period, the contact became stuck in an InQueue state. (ESC-68047)

Admin

  • The employee list file was out of date when downloaded from the Employees page. It was up to date when downloaded from Bulk Upload Portal. Both sources now contain up-to-date files. (UH-39698)

API

  • Some API calls to C700 were incorrectly using http instead of https. (PAPI-6758)

Digital Experience

  • When a contact clicked a rich media link from a message in a closed asynchronous chat, the action failed. (DE-89894)

  • When an agent added an attachment to a chat message, DFO would sometimes include the attachment in the next message as well. (DE-109197)

  • Quick response for outbound emails showed a blank table instead of a populated one. (DE-108061)

CXone Recording

  • There was an issue filtering certain teams in Interactions. (CXREC-64518)

CXone Studio

  • When a script was in read-only mode, actions could be moved after a web editor was opened and closed. (CXWS-15649)

Pipeline Update 158

Published 18 June 2024*

ACD

  • In advanced routing environments that transferred contacts to PCI Pal for payments, contacts sometimes failed to transfer back to the CXone agent. (AOR-5095)

Admin

  • In employee profiles, users only saw the ID of the assigned login authenticator. They could not see the name. (UH-39098)

Digital Experience

  • The Digital Experienceapplication became inaccessible when a case custom field contained hundreds of values. (DE-108099)

  • Deleting Digital Experience forms that were connected to a deleted digital channel returned an error. The error said that the forms were assigned to a digital channel. (DE-107946)

  • When a user set a locale that didn't exist or wasn't supported, the application showed hard-coded values for the UI. Users to not see their custom names. Digital Experience will now translate into English in this scenario. (DE-107601)

  • Rarely, duplicate chats were created intermittently for all agents in an environment. (DE-100554)

  • Weekly and monthly digital evaluations didn't return the expected results. (DE-99005)

  • When a routing queue was configured with a Date and time rule and the brand time zone differed from the server time zone, the setting was enforced in the server time zone. (DE-107782)

IEX WFM Integrated

  • Interval processing would sometimes stop working for certain regions. (INT-47190)

  • Some environments experienced delays in processing. Steps have been taken to improve performance. (INT-47132)

CXone Recording

  • For some agents ACW was not always recorded. (CXREC-61896)

  • In certain instances call recording was not available. The recording alert was "No connection to media server". (CXREC-62950)

CXone Studio

  • Actions could not be dragged to a new location on the script canvas by clicking in the middle of the action's icon shape. (CXWS-16635)
  • After clicking the Actions or API Actions icons in the left icon menu, the corresponding palette did not stop opening. (CXWS-12381)
  • In the Snippet action editor, an error was not shown if the syntax was preceded by any text. (CXWS-17624)
  • In the trace output variables list, hovering the cursor over an item caused flickering , and items were not clickable or selectable. (CXWS-17306)

Pipeline Update 157

Published 28 May 2024*

Admin

  • When an employee was configured with views, navigating to ACD Users could result in a platform error message. (UH-38903)

Digital Experience

  • Agents couldn't reply to a Facebook private message that was more than 24 hours old. (DE-102505)

  • Agents sometimes couldn't reply to contact-initiated Digital Experience SMS interactions. (DE-97327)

Reporting

  • Some agent activity data wasn't included in the Adherence BI report. (INT-46774)

CXone Studio

  • When opening a script that has unsaved changes, clicking the X to close the autosave notification caused the script to open anyway. (CXWS-16937)
  • A long string of text with no spaces in an error message overflowed the red message box in the right panel. (CXWS-13623)
  • Long text overflowed the drop-down box on the Scripts page when the browser language is set to Japanese and the 非アクティブ (Inactive) option was selected. (CXWS-13167)
  • Switching between the Active and Inactive drop-down options caused incorrect records to display on the Scripts page. (CXWS-17342)

24.2 Release

Published 21 May 2024*

Admin

  • Users who had the View permission enabled for CXone Dashboard prior to the 2024-1 release lost access to the page after the 2024-1 release. (UH-37419)

CXone Agent

  • Agents were unable to add a disposition Result assigned by the agent or system at the end of a voice (disposition) or digital (status) interaction. to an interaction when the contact ended the interaction. (AW-25396)

  • For web forms in a screen pop A configurable pop-up window with information about the contact. It appears on an agent's screen after a certain event, usually when the agent connects to a contact., clicking the Submit button resulted in the RunApp-enabled webpage opening in a new tab. (AW-23616)

  • Button clicks had slow responses. (AW-25525)

  • Agents in some environments were unable to transfer or conference calls. (AW-25981)

Bot Builder

  • In certain configurations, a new chat bot sometimes didn't appear on the Bot Builder page. (BB-1829)

Enlighten AutoSummary

  • When an AutoSummary-generated Notes section was edited, the edits didn't appear in the mapped CRM record. (ILLUM-8259)

Data Share

  • The HIERARCHY_COMPOSITION_SCD_DIM table didn't store historical records and HIERARCHY_COMPOSITION_SCD_DIM_VIEW_V001 had filters available that shouldn't have been available. (UH-34871)

  • Certain agent skill Used to automate delivery of interactions based on agent skills, abilities, and knowledge data wasn't available for some environments. (DAT-13987)

Data Streams

  • Data Streams configuration emails sometimes included an invalid business unit ID. (SRE-3464)

  • A record was published more than once if it had the same PoC The entry point that an inbound contact uses to initiate an interaction, such as a phone number or email address. ID configured for multiple destinations. (DST-4674)

Digital Experience

  • The Digital ExperienceCXone Email page sometimes showed the wrong email type for SES email channels. (DE-94809)

  • When an agent clicked the Options button in the chat header, they couldn't close the menu without selecting an option. (DE-92519)

  • When a user went to Digital > Points of Contact Digital, selected a chat channel, and clicked Initialization & Test, the chat widget didn't always load. (DE-93207)

  • Post-chat survey links were opening in the same tab as the conversation. They now open in a new tab. (DE-102520)

  • Chat windows could fail to load when you used the web module analytics API. (DE-102266)

  • Chat window CSS was inherited in areas where it shouldn't have been. This could have a significant impact on the appearance of the chat window. (DE-81514)

Interaction Analytics

  • When you edited a saved search, you had to edit more than one property to be able to click the Save button. (AN-52161)

  • When the top dashboard filter included the same metrics as a widget filter, the dashboard filter overrode the widget filter. (AN-53809)

  • When a dataset had a filter with more than one value, the results disappeared. Removing the filter caused a new filter with the second value to appear. (AN-53340)

Monitoring Gateway

  • In Voice Quality Metrics, the agent and skill names weren't displayed for the chosen contact ID. (PSHM-18397)

platform

  • SSO-enabled users were being logged out of CXone before meeting the idle timeout threshold. (UH-35606)

  • In some environments, users would log in to CXone, refresh, and then be forced to log in again. (SC-49589)

  • Searching for interactions produced an error when the preferred browser language was set to a language other than English. (CXAPP-23409)

Reporting

  • In some systems, opening certain prebuilt reports returned an internal server error. (SC-50464)

  • In certain systems where the hierarchy had changed, the Evaluation Questions and Answer Analysis, Evaluation Question and Answer Details, Evaluation Details, Evaluation Question Details, Evaluator Analysis, and Calibration reports returned the wrong results. (CXBI-5053)

CXone WFM

  • Staffing forecasts were downloaded in the incorrect time zone. (CXWFM-45896)

  • Email notifications for PTO were not received by agents. (CXWFM-46567)
