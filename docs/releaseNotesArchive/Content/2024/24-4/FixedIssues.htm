24.4 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Pipeline Update 168

Published 14 January 2025*

CXone Agent

  • For agents using the CXone Agent WebRTC Extension, the pop-up message prompting agents to enable microphone access appeared only when they first logged in to Agent and when Agent refreshed. In version 1.0.13 of the CXone Agent WebRTC Extension, the pop-up message appears each time the agent is unable to accept a call because they haven't enabled microphone access. This makes the issue clear for agents to identify and resolve. To see this update, download the most current version of the CXone Agent WebRTC Extension from the Chrome web store. (AW-31219)

  • Changes to the WebRTC gateway in ACD Users did not apply immediately. This has been fixed in version 1.0.13 of the CXone Agent WebRTC Extension. To see this update, download the most current version of the CXone Agent WebRTC Extension from the Chrome web store. (AW-31694)

Pipeline Update 167

Published 2 January 2025*

ACD CXone Routing

  • In some systems with advanced routing enabled, contact routing experienced occasional delays. (AOR-6371)

IEX WFM Integrated

  • State synchronization paused when a large number of events were being processed. (INT-49863)

Interaction Analytics

  • Sometimes when a dashboard was created from an old template, the dataset wouldn't load. (AN-58988)
