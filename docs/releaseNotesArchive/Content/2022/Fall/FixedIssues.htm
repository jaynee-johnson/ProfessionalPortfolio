HELP CENTER

Fall 2022 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Cumulative Update 4

Published 3 January 2023*

ACD

  • When users downloaded existing ACD skills, the assigned users column included users who were no longer assigned to the ACD skill. (AUI-17916)
  • In some environments, attempting to rename a point of contact resulted in an error. (AUI-17683)
  • Updating station timeout and refusal timeout values using ACD Users bulk upload failed. (AUI-17665)
  • The audit history of an ACD skill record sometimes showed an inactive user as having made recent changes. (AUI-17197)

API

  • When a WAV file was uploaded with the PutFileStream API with cloud set to on, an error occurred and the upload failed. (FS-2261)

Digital First Omnichannel

  • When a user opened an email to review, approve, or refuse it, the email was stripped of all formatting. (DE-38580)

Security & Compliance

  • Salesforce OIDC users couldn't hit the Authorization Endpoint with the authorization_code grant. (SC-29336)

Studio

  • The Modified Date for a script would change to the current date and time whenever a user opened or closed the script in Studio. (AUI-17673)
  • Saving a script that included the Get Skill Schedule Settings action resulted in an error. (OB-12538)
  • Users on cluster C71 couldn't log in to Studio using the CXOne Gov option. (STX-17026)
  • Studio login sessions were timing out during periods of inactivity. (STX-16914)
  • Users on business units that had been migrated to User Hub couldn't log in to Studio using the CXone Custom Login type. (STX-16840)

Cumulative Update 3

Published 1 November 2022*

ACD

  • Changing an ACD user's Dialing Pattern and saving the change produced a platform error. (MAX-11758)
  • In Omnichannel Session Handling environments, trying to update an ACD user's contact settings produced an error. (MAX-11735)
  • In the Skills tab of ACD user profiles, loading the next page in the table took a long time. (AUI-16962)

Digital First Omnichannel

  • In some environments, adding a quick reply to an email caused the agent application to crash. (DE-37846)

Personal Connection

  • The Reach Percentage on the Skill Control page showed the business unit values instead of the ACD skill values. (OB-12209)

Studio

  • When a script contained a disposition contact - v9 action, it resulted in an error and the script not running. (DE-38385)

Cumulative Update 2

Published 18 October 2022*

API

  • In some environments, /custom API calls involving the CountAgent action didn't return a response. (ORC-18570)

Digital First Omnichannel

  • Sending an embedded link through email resulted in a broken link and an error message for the contact. (DE-35479)

MAX

  • In rare cases, the agent leg didn't connect to MAX. (CX-168593)
  • For Omnichannel Session Handling-enabled environments, cold transferring to another agent failed. (ORC-20075)

Personal Connection

  • When the time zones of a record didn't match between the phone number, city and state, and zip code, the record was not assigned multiple time zones to match them all. (OB-11630)

Platform

  • During a migration to CXone User Hub, the updates to security profiles would sometimes trigger thousands of notifications. (AUI-16936)

Recording

  • When a cluster was processing a large amount of data, the wrong contact was recorded during inbound calls. (MS-5645)
  • Recordings were unavailable for certain environments. (ORC-19340)

Reporting

  • Data Download report 350048 incorrectly showed an SLA value of -1 for some ACD skills. (AUI-17156)
  • The Contact History report didn't show the dispositions assigned to a Workforce Engagement interaction when it was transferred multiple times and had more than two dispositions attached to it. (ORC-18971)

  • Abandoned contacts did not end correctly in some environments. (DAT-6536)

API

Digital First Omnichannel

  • The body of emails didn't always appear in a Studio trace. (DE-35355)

MAX

Fall 2022 Release

Published 27 September 2022*

ACD

  • The business unit Change History tab sometimes shows an unauthorized change to the port count by "Cluster Administrator." (AUI-16779)
  • Changes made to a user profile resulted in a platform error. If you chose to send the report, the changes wouldn't save. (AUI-16090)
  • The Routing Attributes page didn't display the Copy button. (AUI-15056)
  • The Audit History tab would sometimes show events that didn't actually occur. (AUI-16326)

Admin

  • In the employee bulk upload template, setting the Invite column to True didn't send invitations to the users. (UH-20997)
  • When a user opened a role, went to the Change History tab, and clicked More details for an entry, CXone opened in a table of collapsed data that couldn't be expanded. (UH-20978)
  • When users used teams bulk upload to update team channel settings, the voice setting reset to 0. (UH-21200)
  • In some environments, deleted employee profiles would disappear from Employees. ACD Users showed the user as inactive and Audit History attributed the change to user "Impersonated Admin". (UH-21022)

API

  • The /user-management/v1/users/search API call returned a null value for the deletedDate field. (UH-22187)

Bot Builder

  • When users deleted a RegEx from a RegEx entry, the application froze. (BB-753)

Cloud Storage Services

  • Life cycle management rule edits didn't apply to files from the past. (STO-7730)

Developer Portal

  • SkillActivityById and SkillActivity didn't load responses correctly. (IDE-1055)

Digital First Omnichannel

  • In some environments, the contact's response in an interaction didn't automatically appear in the case. (DE-34820)
  • When an agent sent a secured form, the link went to all open cases instead of just the selected one. (DE-34713)
  • The channel filter didn't work when adding a channel from quick replies. (DE-34618)
  • Workflow automation triggers in the WhatsApp channel would sometimes only partially execute or not execute at all. (DE-34010)
  • When an agent removed a resolved chat from their inbox, the contact's window turned gray and said an agent would connect with them shortly. (DE-33762)
  • Adding a new template in the WhatsApp Enterprise channel returned a 400 Bad Request error. (DE-33812)
  • Agents were unable to open gifs and other images received in Instagram direct messages. They also needed to refresh the DFO window after viewing a video attachment to continue using DFO. (DE-33613)
  • WhatsApp templates didn't appear in outbound messages. (DE-33610)
  • When agents opened a case Approval tab and then clicked the Approve by button, elements in the form overlapped each other. (DE-33073)
  • Emailing a transcript would show as successful, but then a 500 error appeared and the contact didn't receive the transcript. (DE-32805)
  • Contacts couldn't change their names. (DE-32594)
  • In some environments, agents would receive messages from LinkedIn but couldn't reply to them. (DE-32471)
  • Rarely, forwarding a case opened an error page. (DE-31962)
  • In some environments, chat windows were blank for the agent. (DE-31642)

Interaction Analytics

  • Transcriptions for transferred interactions with multiple segments only showed the start of the interaction. (AN-38498)
  • Dataset changes took too long to process. (AN-37367)
  • Transcripts of transferred chats were duplicated. (AN-36951)

PartnerHub

  • Presence Adapter didn't sync user presence in MS Teams. (PIN-4420)
  • In some environments, upgrading from Dir1.0 to Dir2.0 resulted in users not being able to set up a directory. (PIN-4311)
  • When an agent took an inbound call in an environment with bi-directional Presence Sync enabled, the agent state in MAX didn't change. (PIN-4102)

CXone Recording

  • In some environments, inbound calls would show as unavailable for playback, and the voice recorder would sometimes be unreachable. (STO-7772, STO-7763)
  • In rare cases, recording would continue for hours after a call ended. (ORC-17115)

Reporting & Dashboards

  • In some environments, accessing Prebuilt Reports resulted in an internal server error. (DAT-5248)

Security & Compliance

  • Some environments were encountering a 500 errors related to OpenID tokens. (SC-25447)

Virtual Agent Hub

  • In DialogFlow CX, the agent environment couldn't be updated after a conversation profile was created. (AAI-3822)
  • DialogFlow CX sometimes returned intermittent 500 errors. (AAI-3453)
  • The PromptAndCollectNextResponse branch was always taken, even when the UserInputNotUnderstood branch should have been used. (AAI-3451)
  • DialogFlow CX sometimes cached participant client and not the GRPC channel. (AAI-3339)

Workforce Intelligence

  • When users tried to access the WFI Rules page, they were rerouted to Reporting Dashboards. (API-9682)
