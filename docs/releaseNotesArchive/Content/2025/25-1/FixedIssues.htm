25.1 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

25.1 Cumulative Update 8

Published 22 July 2025*

Agent

  • When an agent used a dialing skill with a high dialing ratio, call control sometimes showed the wrong contact for several seconds. (AW-42126)

Digital Experience

  • When an agent logged out our the agent application and then quickly logged back in, the digital contacts they were assigned before logging out were still assigned to them. (DE-134285)

Reporting

  • Old voice contacts sometimes appeared in the Active Contact widget. (VOL-7005)

25.1 Cumulative Update 7

Published 24 June 2025*

ACD

  • Bulk uploading stations failed with an error about an unknown column header when the Email Refusal Timeout field was present. (AAD-34217)

  • Opening a point of contact and saving without making changes triggered a validation error requiring a DNIS. (AAD-34097)

ACD Routing

  • In certain systems, old disconnected calls were sometimes being routed to agents. (VOL-6819)

  • Agents were receiving voice and digital interactions simultaneously when Channel Lock was enabled. (AOR-7768)

Admin

  • Under certain conditions, teams that were enabled for Performance Management didn't appear in the Performance Management feeds. (AAD-34365)

Agent

  • When an agent was handling a case in Agent for Salesforce and had another case in their inbox, unassigning the case in the inbox closed the case instead of rerouting it. (AW-41031)

  • The page navigation for the interaction history table didn't work correctly. (AW-40871)

  • When an agent added an image between two paragraphs in the body of an email and then deleted the image, the image still sent as an attachment to the email. (AW-40725)

  • The commitment calendar showed the wrong day of the week if the agent's browser language reflected a country where the week start is on a different day. (AW-40617)

API

  • It was possible to send two ContactHangupv15 requests for the same contact when using a third-party tool. This resulted in a blank Contact End Reason in reports. (AW-40673)

Attendant

  • User Availability was incorrectly updated to In Office during bulk upload, even when it was previously set to Out of Office. (ATT-19586)

Studio

  • Users couldn't use V17, V20, V22, or V31 of the 'Get a team's agents' framework action in Studio. (AAD-34116)
