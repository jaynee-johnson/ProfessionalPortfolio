25.2 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Pipeline Update 176

Published 15 July 2025*

Copilot for Agents

  • When the prompt editor API failed, the Manual Query Prompt drop-down contained unexpected values. (CSA-30064)

Reporting

  • Scheduled custom reports sometimes included the wrong date in the header. (RPT-47228)

Pipeline Update 175

Published 24 June 2025*

Digital Experience

  • When an agent changed the value of a custom field, the update disappeared when the contact sent a new message. (DE-133147)

  • Security has been enhanced for chat bots. (DE-130929)

  • When an agent transferred or tagged a case, the case history showed "API assigned" instead of the agent's name. (DE-129289)

  • When a contact initiated a second chat on the same webpage they had started a previous one, the custom field data wasn't included in the new chat session. (DE-135571)

Knowledge Hub

  • Sometimes synchronization completed without the status changing from Syncing. (CSA-28877)

Pipeline Update 174

Published 3 June 2025*

ACD Routing

  • In certain configurations, contacts that were supposed to route to a specific agent routed to other agents. (AOR-7520)

Admin

  • When more than 20 users were assigned to a role, the assigned users list wouldn't scroll past the first 20 users. (SC-68878)

Digital Experience

  • Security in the chat bot has been improved. (DE-130929)

  • When agents added tags to a case, the case history showed "API assigned" instead of the agent's name. (DE-129289)

Interaction Analytics

  • Data in the AutoDiscovery tool sometimes didn't load. (AN-62708)

Reporting

  • Custom report schedules sometimes sent blank reports. (RPT-47393)

Pipeline Update 173

Published 13 May 2025*

ACD Routing

  • Digital contacts were not being assigned automatically in certain regions. (AOR-7195)

Digital Experience

  • Contacts sometimes took up to 10 minutes to route. (DE-130709)

  • The removeContactCustomField JavaScript API did not properly clear custom fields from local storage, resulting in stale data being displayed after refresh. (DE-129736)

  • Custom JavaScript commands configured in the chat channel setup were not executing. (DE-129339)

  • Agents were being assigned contacts for skills they were not currently active in. (DE-128553)

Interaction Analytics

  • Workspace templates were not displaying in the workspace drawer, preventing users from adding templates to datasets. (CXIA-3694)

  • In certain configurations, top company profile widgets didn't display data from all included dates. (AN-63656)

Reporting

  • Users experienced delays in report execution during peak hours, causing reports to queue and complete later than expected. (RPT-47009)

Studio

  • The GET Contact Custom Fields action returns a variable called Customer Fingerprint. The IP address returned with it was always null. (DE-129130)

  • When performing copy and paste or cut and paste actions in a script, an error occasionally occurs, preventing the script from being saved. (CXWS-29796)

  • When a user clicks on a breakpoint in the snippet editor, it is displayed in gray instead of red. (CXWS-28799)

  • When a new trace event is received, the list of trace events scrolls up automatically. (CXWS-28798)

  • Clicking on the Name column to sort the table in the Trace Output window does not work. (CXWS-28792)

  • When a connector is dragged to an action that is far away from the visible canvas, the connected actions become invisible. (CXWS-28423)

  • When attempting to save a script that contains a SNIPPET action with a caption that includes line breaks, the "An unknown error has occurred" error message is displayed and the script is not saved. (CXWS-28422)

  • When attempting to view an old version of a script that has unsaved local changes, the unsaved version is displayed in edit mode instead of the old version in view-only mode. (CXWS-28192)

  • The Email category is hidden in the Actions palette when opening an Email script. (CXWS-27924)
