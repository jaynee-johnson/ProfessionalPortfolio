Spring 2023 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Cumulative Update 6

Published 16 May 2023*

ACD

  • When an agent was forced to disconnect, the end reason was "Not Applicable". (ORC-21072)
  • Opening ACD > ACD Configuration > Business Units sometimes produced an error. (AAD-20383)

IEX WFM

  • Intervals would sometimes go missing in CXoneWFM products, including IEX WFM. (INT-41426)

MAX

  • The new Hide Caller Phone Number permission didn't work correctly; with it enabled, agents were still able to see the caller ID of their contacts. (MAX-12553)

Media Server

  • The media server sometimes returned a "No connection to media server" error during conference calls that were being recorded. (ORC-23491)

Personal Connection

  • When users uploaded a calling list with their browser language set to German, the batch expiration time changed from PM to AM. (OB-14741)

CXoneWorkforce Management

The Groups and Skills filters in the Meeting Scheduler were not functioning. Users can now see employees based on the Groups and Skills filters in the Select Employees modal. (CXWFM-36191)

Cumulative Update 5

Published 25 April 2023*

An improvement to MAX accessibility has been released with this cumulative update. For details, see Cumulative Update Features on the Spring 2023 Release Notes page.

ACD

  • Clicking Station Profiles in the navigation pane produced an error. (AUI-19914)
  • Users experienced long delays when saving updates to skill assignments in both ACD Skills and ACD Users. (AUI-19637)
  • Some ACD User settings were updated by the CXone system during an upgrade. (API-11278)

IEX WFM

  • IEX WFM reported incorrect call volume numbers that were higher than they should have been. (INT-41134)

Personal Connection

  • Agent-specific calls were sometimes not being routed to the correct agents. (OB-14083)

  • With a predictive dialing ACD skill, an agent could add an unlimited number of calls to a conference. Overloaded conference calls resulted in cluster timeouts. (OB-14019)

Studio

  • In Personal Connection skills where the script included the GetProspect action, the original agent was being returned instead of the prospect details. (OB-14209)
  • The script history sometimes showed duplicate date and time entries. (AUI-19638)

Supervisor

  • Supervisor didn't always show the correct agent state. This could make it look like queued interactions weren't routing to available agents. (ORC-21044)

Cumulative Update 4

Published 4 April 2023*

Enhancements to DFO quick replies have been released with this cumulative update. For details, see Cumulative Update Features on the Spring 2023 Release Notes page.

ACD

  • Rarely, a disconnected contact would route to an agent. (ORC-22550)
  • In certain environments, a daylight savings time change caused a "Contact already routed" error to occur every hour. (ORC-23235)

API

  • Calling the services/v26.0/interactions/{contactId}/signal? API using an invalid contact ID produced a 500 error instead of a 404 error. (VOL-1113)

  • The v18.0/contacts/states API returned both active and completed contacts, which increased the value of totalRecords. (RPT-27277)
  • When the state field was updated using the v24 or v25 PUT /Agents API, the address field updated but the state field did not. (AUI-19499)
  • GET /agents/<agentID> and PUT /agents were returning different values for the terminationDate. (API-10999)

Digital First Omnichannel

  • New channel names of a certain length and within the character limit failed to save. (DE-39430)

MAX

  • Some agents were being logged in and out automatically after they had already logged out and were no longer working. (MAX-12129)
  • When an inbound email was initiated with a Studio PopUrl action, the email page opened in a new external browser window. (MAX-12109)

Partner Tools

  • Erasers requested through the GDPR request page retained some interaction data. (OSCI-396)

Personal Connection

  • The downloaded list of attempts would show duplicate rows with different agent IDs. (OB-14039)

  • The downloaded list of prospective contacts showed the Modified Date for a change but didn't reflect the actual change. (OB-13327)

Platform

  • When a user account contained Japanese characters in the First Name or Last Name, edits to the employee profile didn't sync with the ACD user profile. (API-11037)
  • For some environments, visiting the legacy dashboard feature in Reporting, logging out and logging back in resulted in all icons disappearing from the left navigation pane. (AUI-19548)

Reporting

  • The List of POCs Expanded data download report showed data from other business units. (RPT-27943)
  • The outbound store Contact Detail report didn't show data in the Disposition column. (OB-14123)

Studio

  • Attempting to save scripts that contained the Set Transport Code action resulted in an error message. (RPT-26539)

Cumulative Update 3

Published 14 March 2023*

ACD

  • Some environments encountered slow response times while opening ACD skill profiles in CXone. (AUI-19168)
  • The business unit Audit History tab showed that changes were made by Admin_do_not_use when the ID of the user who made the change was unavailable. It now shows the user as Not Applicable. (AUI-19061)
  • Some environments received a platform error when trying to add a disposition to an ACD skill. (AUI-18966)

CXone Attendant

  • CXone Attendant sometimes showed empty voicemail messages with no create date, source, or contact ID. (ATT-11939)

Digital First Omnichannel

  • The Agent Contact Duration reports sometimes showed missing or incorrect times for Handle Time, Active Time, and Inactive Time. (DE-41917)

IEX WFM Integrated

  • Reported contact counts were inaccurate when emails and a call were handled at the same time. (INT-39968)

MAX

  • Agents who manually entered a number and pressed Enter twice no longer had their calls dialed automatically. (MAX-12276)

Personal Connection

  • The downloaded list of attempts would sometimes include data from other business units. (OB-13980)

  • Clicking Play Message in ACD Skills > <OBSkill> > Parameters > Answering Machine Detection Options sometimes produced a "file does not exist" error message, even if the file was valid. (OB-13839)

Platform

  • Users were being logged out of CXone after 15 minutes, no matter what the timeout setting was. (SC-34985)

Reporting

  • Attempting to sort a custom report template by point of contact name failed to save. It produced no error message. (RPT-25829)

Studio

  • Scripts using a LineType variable showed misclassified lines in the trace data. Landlines were sometimes called wireless and vice versa. (OB-13608)
  • When a marquee was set up for MAX using the Messages action, and both the duration and expire time were set, the marquee didn't appear in MAX. (ORC-21541)
  • The ContactTakeover action sometimes ended the wrong contacts. (ORC-20056)

API

  • The list of unassigned ACD skills included agentless ACD skills. (API-10669)

MAX

Spring 2023 Release

Published 24 January 2023*

ACD

  • The business unit section called API Daily Limit of Method Calls didn't specify which type of API it controls. The section heading has been changed to SOAP API Daily Limit of Method Calls. (AUI-17368)

Admin

  • Under certain circumstances, teams couldn't be added to roles. (API-10299)
  • Rarely, an organization would be charged for a non-billable user. (AUI-18564)

Agent Assist Hub

  • In certain environments, Agent Assist Hub wasn't loading in the Integrations and Admin UI in Studio. (RTIG-3414)

API

  • Running a GET query for /agents/{agentId}/login-history required the orderby field. (RPT-24380)
  • Contact details from the DFO API didn't correctly map the custom case field types list or tree. (DE-41084)

CXone Attendant

  • Changing the Release Upon User Deactivation field, located on the CXone Attendant tab of the business unit settings page, made changes to all existing CXone Attendant users without a notice. A message window now notifies you that changes to all your CXone Attendant users will occur. (ATT-9530)

Cloud Storage Services

  • When two long-term file retrieval jobs with the same criteria were created in quick succession, the second job became stuck with the RestoreAlreadyInProgress status. (STO-8409)

CXone QM

  • The user was unable to evaluate when the Perform Evaluations permission was enabled on the secondary role.

CXone Recording and Interactions

  • An intermittent issue was causing an internal process to fail for some interaction recordings. (DL-4864)
  • Some recordings were several hours long. (ORC-20692)

  • Some interactions that had dispositions applied to them showed no value for Disposition on the Interactions page. (OB-13056)

  • ScreenAgent was not working with a proxy configuration. (CXREC-24449)

CXone Workforce Management

  • The Special Days calendar and the displayed dates weren't matching. (CXWFM-31206)
  • New users could not log in to the EM Agent mobile app. (CXWFM-33135)

Digital First Omnichannel

  • Clicking the WhatsApp Message Templates drop-down highlighted a gray box instead of opening the drop-down. Clicking the gray box opened the drop-down. (DE-41102)
  • All messages sent to the sentiment analyzer were evaluated as negative. (DE-41096)
  • The date range for the DFO thread search didn't work if more than one channel was selected. (DE-40443)
  • Outbound emails were marked with the wrong start time. (DE-40307)
  • Some contacts in WhatsApp were blocked from sending outbound messages. (DE-39872)
  • If an agent closed the chat and clicked Back to Conversation, they could enter text and click buttons in the chat window. (DE-39732)
  • DFO routing was delayed during times of high traffic. (DE-39708)
  • Clicking ACD > DFO > Hours of Operation redirected users to the ACD Users page. (DE-39645)
  • When an agent changed the From address for a reply, the approved case was sent with the original From address. (DE-37944)
  • When an agent replied to an email contact, the branded template looked distorted. (DE-37375)
  • Permissions could be edited in ACD > DFO > Roles, which is not being audited. (DE-36680)
  • The Email for Transcripts button would sometimes error out. (DE-36179)

Interaction Analytics

  • Adding a category filter to a data set resulted in the sentiment bar disappearing. (AN-41935)
  • Users with AutoDiscovery couldn't filter by keyword. (AN-41783)
  • Users whose employee profiles weren't marked as Can be Analyzed were still being analyzed. (AN-34671)
  • Resampling a dataset would sometimes execute for long periods without completing. (AN-37293)

MAX

  • The panel didn't open automatically when called using the RunApp Studio action. (MAX-11696)
  • MAX would sometimes report ScreenAgent as recording when it had actually timed out. (MAX-10990)
  • The Fax override button didn't appear in MAX when configured properly in ACD. (MAX-11105)

PartnerHub

  • When an error 500 occurred, Log Reader didn't include that line. (PSHM-12789)

Personal Connection

  • Dialer calls were dispositioned as Error by System when the script contained the NC Placecall action with certain branches configured. (OB-12157)
  • Agents were incorrectly receiving inbound phone calls while logged in to a Personal Connection skill. (OB-12758)
  • Callbacks that occurred when the assigned agent was busy were not scheduled according to the specified attempt times. (OB-12568)

Platform

  • Agents would sometimes appear to be available on the WFM > Real Time Adherence page after logging out of CXone. (OB-13051)
  • Using a token endpoint with access keys would sometimes result in a 401 error. (SC-29114)
  • Modifications to employee profiles weren't always syncing to the corresponding ACD user profile. (API-10482)

Reporting

  • When a user created a new field on the Business Data page, and the new field had a display name containing 128 characters, the Evaluation Details and Evaluation Question and Answer Details reports didn't include some data. (DL-4907)

Studio

  • Running a script with the Dynamic Address book entries V15 action would sometimes leave an internal server error in the trace file. (API-9970)

Supervisor

  • ACD skill names with special characters didn't appear correctly in Supervisor. (API-10296)

Tenant Management

  • The Impersonate and Support features couldn't be toggled off. (TM-6414)

Virtual Agent Hub

  • When a large number of requests were made, the requests would begin to fail. (AAI-4270)
  • Dialogflow fulfillment didn't play the entire audio response. (AAI-4616)
