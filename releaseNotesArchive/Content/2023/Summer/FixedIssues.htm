HELP CENTER

Summer 2023 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Cumulative Update 5

Published 29 August 2023*

ACD

  • Points of contact no longer supported the pound symbol (#) in the name. (ORC-27129)

  • When an agent had an email interrupted by a call, completed the call, and then completed the email, they couldn't change their state from ACW. (ORC-25803)

  • When an administrator tried to turn everything off in the team Contact Handling settings while in single contact mode, an error appeared. (AAD-22123)

  • In some environments, adding or editing dispositions to an ACD skill caused the system to time out. (AAD-22038)

API

  • In v27.0, the agents/state-history call returned a 500 error. (RPT-31948)

Digital First Omnichannel

  • In rare cases, a digital skill would have an Audit History Modified By value of Not Applicable. (DE-60024)

  • Sometimes the skills in Supervisor showed long wait times or counts that didn't match the numbers in DFO. (DE-59239)

  • When cases weren't being properly closed out, the growing list of active contacts slowed down processing. (DE-57975)

Personal Connection

  • System-applied dispositions sometimes changed to No Disposition, especially for calls with a short duration. (OB-16151)

  • XS scripting stopped correctly mapping the source to the outbound skill. It returned the error "The mapped destination type(s) does not match those allowed for the skill." (OB-15781)

Reporting

  • In the Contact History report and on the Interactions page, some outbound calls were missing their call recordings. (DAT-11634)

  • Some contact IDs didn't appear in the Contact History report. (DAT-10898)

Studio

  • The Whisper action caused errors and resulted in crosstalk and dropped calls. (MS-9335)

  • In scripts run for MAX, the Indicate action couldn't enter the Off state. (MAX-12981)

  • For DFO chat, internal events began happening in a different order, causing the OnAssignment action to not trigger as expected. (DE-59961)

  • In scripts for Personal Connection ACD skills, the WAIT action was sometimes ignored, causing all interrupting contacts to be stuck in queue. (ORC-26938)

Pipeline Update 5

Published 29 August 2023*

CXone Recording

  • Interactions stopped showing agent names. (CXREC-43412)

  • The Millisecond value for StartTime for call wrap up was too long. (CXREC-42972)

Digital First Omnichannel

  • Having too many hours of operation in an environment produced errors. (DE-62480)

  • The MAX version number overlapped with the "Waiting customer contacts" message. (DE-61753)

  • Delayed conditions weren't working. They produced the error "One or more fields are invalid". (DE-61098)

  • Images were removed from email replies. (DE-61096)

  • For agents who had multiple routing queues, transferring a case only showed five possible users even though the default limit is 30. (DE-60956)

  • In MAX, inbound digital emails would arrive with their attachments missing. (DE-60913)

  • Rich text HTML quick responses were available to use in plain text chat channels. (DE-57270)

Cumulative Update 4

Published 8 August 2023*

ACD

  • Adjusting the settings in the ACD Users Contact Settings tab sometimes produced an error. (AAD-21491)

  • Creating an hours of operation profile with spaces in the name produced an error instead of accepting it. (AAD-21536)

  • Rarely, inbound calls weren't routing to agents who were handling emails in a n OSH environment. (ORC-26251)

API

  • GET /skills/{skillId} failed for both v24.0 and v26.0. (DE-58693)

  • Contacts/Completed showed duplicate contact IDs. (OB-15697)

  • Contacts/Active didn't produce the same data in the latest version of the API as it did in earlier versions. (RPT-31099)

Digital First Omnichannel

  • Recordings for digital interactions didn't show the name of the associated team. (DE-53570)

MAX

  • The Agent Performance report didn't work when an agent selected a custom range for the report data. (MAX-12948)

  • Sometimes contacts were automatically refused when they shouldn't have been. (ORC-26219)

  • The app would crash for agents who were in an available state after they were inactive for a while. (ORC-26597)

Platform

  • Sometimes when supervisors would force agents to log out, the agent remained logged in. (ORC-25857)

Reporting

  • The Admin and Agent CXone Login data download report didn't show updates to locations. (AAD-21051)

Studio

  • Typing '(' in the Snippet Editor produced an error. (STX-19817)

Pipeline Update 4

Published 8 August 2023*

CXone Recording

  • The skill names of agents in a conference call were getting interchanged. (CXREC-41658)

  • The DoNotRecord policy was being ignored. (CXREC-41068)

  • Segments of recordings were missing with a recording alert message, No Connection to Media Server. (CXREC-40545)

Digital First Omnichannel

  • Exporting cases to XLXS or CSV didn't work. (DE-58967)

  • Contacts weren't seeing rich text on their side of a live chat. (DE-58293)

  • When an agent changed the recipient of an email, sent the message as pending, and then confirmed they wanted to send the message to an unknown contact, the case was saved as resolved. (DE-59299)

CXone WFM

  • CXone WFM Schedule Manager prompted Jan 2024 when selecting week of Jan 2023. (CXWFM-36856)

Cumulative Update 3

Published 18 July 2023*

ACD

  • In OSH-enabled environments, agents periodically experienced call delivery failures. (ORC-23202)

  • When a script transferred a call to another skill, and that script included the LogRecordingPro Studio action, the wrong SIP endpoint was assigned to the call. (ORC-24969)

  • When contacts initiated a CXone-powered chat from a company's website, allowed the chat to time out, and used the same browser tab to navigate to a different company's website and initiate another CXone-powered chat, the history of the first chat appeared in the new chat window. (CHT-5106)

  • For chat ACD skills that use a v2 chat profile and an embedded script, the contact's pop-up window for the chat was blank. (CHT-5045)

Personal Connection

  • The Details tab of a Personal Connection skill didn't show the assigned DNC group. (OB-14651)
  • In very few environments, the dialer would sometimes fail. (ORC-26074)

Reporting

  • The Performance Management Agent States report always showed a duration of 0. (RPT-30740)

  • The Monitored Calls data download report was blank. (OB-15485)

Studio

  • The Snippet editor didn't support reserved characters like periods and parentheses. Using these characters produced an error message. (STX-19511)

  • When a user exported a trace file and viewed it in a text editor, redacted variables were not hidden. (STX-19513)

Admin

CXone Recording

CXone WFM

Digital First Omnichannel

  • Agents could send a new message in a closed case. This created a new case that wasn't routed. (DE-13530)

Reporting

  • The Change Audit report returned an "Internal Server error" message. (CXBI-3954)

Tenant Management

  • When two tenants shared the same name, they couldn't share the same allowed domains. (UH-27410)

Summer 2023 Release

Published 30 May 2023*

ACD

  • Switching to OSH sometimes caused contacts to not route to agents for a period of time. (ORC-22984)
  • Inbound emails would sometimes become stuck and not route to agents. (ORC-22908)
  • Parked emails would sometimes end when the agent logged off, and they wouldn't reroute. (DAT-7177)
  • Uploading a WAV file using the Browse ACD Files page would sometimes produce a 403 error. (FS-4298)
  • Removing holidays from an hours of operation profile failed. (AAD-19246)

Admin

  • Entering an ACD Login Id for an employee profile didn't save correctly. (UH-24100)

Agent Assist Hub

  • The audio recording started late and sometimes the recordings failed to save. Agent Assist Hub didn't set the status of the interaction to Complete when it was over. (AAI-4594)

CXone Agent

  • Fallback text wasn't displayed to contacts if rich messages couldn't render. (AW-7163)

CXone Agent Embedded

  • Fallback text wasn't displayed to contacts if rich messages couldn't render. (AW-7163)

CXone Agent for Microsoft Teams

  • Fallback text wasn't displayed to contacts if rich messages couldn't render. (AW-7163)

CXone Agent Integrated

  • Fallback text wasn't displayed to contacts if rich messages couldn't render. (AW-7163)

CXone WFM

  • The Groups and Skills filters in the Meeting Scheduler were not functioning. Users can now see employees based on the Groups and Skills filters in the Select Employees modal. (CXWFM-36191)

Developer Portal

  • The example in Admin API for uploading files was incorrect. (IDE-1574)

Digital First Omnichannel

  • The screen reader didn't let users know when a chat message was sent and received. (DE-51653)
  • A field called URL for transferring customer from other channels to Live Chat was available in Points of Contact Digital > Chat Settings > Basic even though it's not supported. (DE-46181)
  • In inbound emails that included recipients other than the DFO point of contact, the other recipients weren't visible to the agent. (DE-45853)
  • Devices running Android 7 OS failed to connect to the CXone Mobile SDK chat back end. (DE-44882)
  • When an agent would copy content from certain external apps and paste them into an email, DFOMAX would become disabled and refresh. (DE-44098)
  • For some contacts, the Agent Contact Duration reports showed missing or incorrect times for Handle Time, Active Time, and Inactive Time. (DE-41917)
  • Agents could send a new message in a closed case. This created a new case that wasn't routed. (DE-13530)

MAX

  • Logging in to MAX failed for users with accented characters in their name. (MAX-12376)
  • The Performance report would show mismatched inbound and outbound call data, resulting in the performance percentage going above 100%. (MAX-10999)
  • When agents would schedule a call, MAX would sometimes offer the wrong time zones. (MAX-10980)

PartnerHub

  • Rarely, a Port Management request would incorrectly be marked as under review, which prevented a new request from being made. (PSHM-13562)
  • Starting a new request in the Log Reader produced an error. (PSHM-14468)
  • In the Oregon region, no Directory Sync region showed for Microsoft Teams. (PIN-3894)

Personal Connection

  • Calls sometimes produced hundreds of CTI events, which caused recording to fail. (ORC-22885)

Platform

  • Some environments couldn't access files on the FTP server. (FS-3569)
  • The Contact History report and the Interactions page sometimes showed conflicting values for the duration of chat interactions. The value on the Interactions page was incorrect. (ORC-23834)

Reporting

  • Attempting to sort a custom report template by point of contact name failed to save. It produced no error message. (RPT-26801)
  • In the Agent Contact Performance prebuilt report, Inactive Time would incorrectly report 0. (DAT-9526)
  • The Evaluation Question Details report failed to generate when the Answer column was included. (CXBI-3235)
  • The Evaluation Question and Answer Details report was returning an internal server error. (CXBI-3150)

Studio

  • The Modified Date for a script would change to the current date and time whenever a user opened or closed the script in Studio. (AAD-16125)
  • The TTSVoice action duplicated the previous text for two-byte characters. (AAI-5743)

CXone Virtual Agent Hub

  • In rare instances, Google Dialogflow ES setups experienced issues with bots not answering. (AAI-6119)

  • Spanish integrations of Google Dialogflow ES didn't recognize the word "Fax." (AAI-6077)
