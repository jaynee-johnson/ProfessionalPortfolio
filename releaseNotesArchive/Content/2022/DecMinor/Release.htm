December 2022 Minor Release Notes

This page has been updated to show the final products and features in the December 2022 Minor release of CXone. Features are subject to change between now and final deployment.

How to Use This Page

All new products and features are shown by default. If you want to see only products and features in your CXone system, you can clear the checkboxes to remove features you don't need from your display.

Each feature description includes a table that states whether the following details apply to it. means that the detail applies to the feature and means that the detail does not apply to the feature.

Customer Request The feature was developed by customer request.
UI Change The feature introduces major user interface (UI) changes.
Availability

When you can expect to see the feature in your environment.

  • On toggle— This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications , watch for one that indicates when that is.
  • On deploy—The feature is immediately available when the new release deploys to your environment.

Admin

CXone Supervisor Permission

Previously, the Supervisor permission (under Supervisor > General Permissions) was linked to the Launch Supervisor permission (under ACD Permissions > Supervisor). This means that if you turned one on or off, the other permission would automatically change as well.

Starting this release, both the Supervisor permissions are independent of one another.

Customer Request UI Change Availability On toggle This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications, watch for one that indicates when. See the Service Notification Timeline page in the CXone online help.

Bot Builder

Folders

You can organize intents and entities into folders. This experience also applies to stories and rules. Any existing organization you have for stories and rules is automatically converted to the new folder structure.

Customer Request UI Change Availability On toggle This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications, watch for one that indicates when. See the Service Notification Timeline page in the CXone online help.

Bot Skills

You can categorize things your bot can do into bot skills. These are not the same as the skills used in ACD. You can add any folder or API integration to a skill. This means bot skills can contain stories, rules, intents, entities, forms, or full API integrations. You can filter the view of your data by each bot skill.

Customer Request UI Change Availability On toggle This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications, watch for one that indicates when. See the Service Notification Timeline page in the CXone online help.

CXone Supervisor

New Skills Tab

There is a new Skills tab on both the Insights and Live Monitoring pages. The Skills tab has replaced the Queue widget on the Insights page.

Using this tab, you gain better visibility into your agents' skills. This enables you to optimize your queues efficiently.

Customer Request UI Change Availability On toggle This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications, watch for one that indicates when. See the Service Notification Timeline page in the CXone online help.

Modify Agent State

You can now modify an agent state to properly reflect their current status. This ensures that the contact queue is handled correctly.

To change the agent state, you must enable the Modify Agent State permission. In CXone, go to Admin > Security > Roles and Permissions > Permissions. The permission is under Supervisor.

Customer Request UI Change Availability On toggle This is a minor release. The feature should be toggled on at the time of deployment. If you've registered for Service Notifications, watch for one that indicates when. See the Service Notification Timeline page in the CXone online help.

Studio

Changes to Categorization of Actions on Framework Tab

Changes will be made to the Framework tab in Studio. Some category folders will be renamed. Others will be consolidated to reduce the total number of categories. Additionally, all actions on the Framework tab will be located inside category folders. The Filter feature on the Framework tab will not be impacted by the changes. These changes will simplify the Framework tab and make it easier to find the actions you're looking for.

The Framework tab is not visible in Studio by default. It must be enabled by your CXone Account Representative.
