HELP CENTER

Summer 2022 Fixed Issues

The date provided for each update is the date it was ready for release. Watch for a notification about when your organization will receive it. It will generally be within two weeks of the given date.

Cumulative Update 5

Published 6 September 2022*

ACD

  • Emails parked for an extended period wouldn't reroute. (ORC-17909)

Dashboard

  • The app would freeze for long periods of time. (SUP-4180)

MAX

  • The Performance report showed inaccurate data that didn't match the Contact History report. (RPT-21734)

Personal Connection

  • The time zone Hawaii Standard Time (HST) wasn't mapping to records in the dialer. (OB-11262)

Platform

  • Users were sometimes logged out of CXone unexpectedly. (AUI-16651)

Recording

  • Recordings from transferred calls were sometimes unavailable for playback. CXone returned a system error when this occurred. (ORC-18265)

Reporting

  • The value for certain metrics in the Outbound Agent Performance report didn't match the value of the same metrics run in a custom report for the same time period and interval. (RPT-21564)
  • The Outbound Report Store Contact Details report showed either the wrong agent or no agent associated with calls where the call was force disconnected. (RPT-21423)

Studio

  • The Framework action Dnc Records by groupId caused errors on save. (API-9699)
  • Manual outbound SMS scripts sometimes wouldn't save. (RPT-21421)
  • When a user included the 'Time' variable in a Snippet action via the String Editor or Eclipse on Data Properties, it saved as lowercase 'time.' (STX-15731)

Supervisor

  • The app would freeze for long periods of time. (SUP-4180)

CXoneWorkforce Management

  • The Forecast Profile page slowed or crashed when multiple values for the By Time shrinkage option existed. (CXWFM-28483)

  • Real Time Adherence displayed some mapped activities as unmapped (CXWFM-30398)

Cumulative Update 4

Published August 16, 2022*

ACD

  • Assigning a point of contact to a DNIS configured for the tenant returned an error. (CX-166084)
  • Email and work item contacts would get stuck in queues. After the contacts were ended, they remained in the PersistentContactLog. (ORC-17725)
  • In some environments, it took a long time for inbound calls to connect to an agent. (ORC-18152)
  • In an Omnichannel Session Handling-enabled environment, agents couldn't transfer a call when they had another interaction open. (DAT-6074)
  • Changes made to an ACD User profile resulted in a platform error. The changes wouldn't save if the user chose to send the error report. (AUI-16090)
  • When a station ID assigned to agents was deleted, those agents' ACD user profiles became inaccessible. (MAX-11113)

CXone Attendant

  • In the ACD Users CXone Attendant tab, inactive profiles included the Edit button and didn't show a CXone Attendant Access Type of None. (ATT-9674)

MAX

  • In environments where a webpage opened when an agent took a new call or chat, browser refresh issues on those webpages sometimes occurred. (MAX-11094)

Personal Connection

  • When users tried to save a Personal Connection ACD skill with Override Caller ID disabled, CXone returned a "caller ID required" error. (OB-11208)

Reporting

  • Agents with the Contact HistoryAgent View permission enabled were able to see contact history reports generated by any agent instead of only their own. (RPT-21415)
  • In some environments, users couldn't search for agent names in the Contact History report to find contact IDs. (RPT-21408)
  • Opening a Contact Details report from the Contact History report sometimes caused an error. This happened when the agent who handled the interaction was on a team with a name that exceeded 30 characters. (RPT-21197)

Studio

  • The REST API action ignored the specified TimeOutInMilliSeconds. Instead, it always timed out after 30 seconds. (ORC-17313)
  • Script histories showed incorrect data. (AUI-16125)

Cumulative Update 3

Published July 26, 2022*

ACD

  • Removing an ACD user's Middle Name failed to save. (API-9443)
  • The Audit History tab of some features sometimes showed entries for events that didn't happen. (AUI-16130)

API

  • When /skill/summary and /skill/{id}/summary requests were sent with only start date and end date parameters, a 500 error was returned. (RPT-21053)
  • GET agents/state-history didn't return milliseconds for the StartDate field. (RPT-20791)
  • PUT points-of-contact/{pointOfContactId} didn't set the specified chat profile value. Instead, it set the default chat profile. (API-9529)

Customer Chat Interface

  • URLs sent to contacts through chat showed up as plain text without a preview. (CHT-4331)

CXone Attendant

  • The side menu would sometimes disappear. (ATT-9504)

Reporting

  • Running a custom report manually or on a schedule failed and returned a platform error. (RPT-21033)
  • Data Download report 525, List of POCs Expanded, sometimes contained data for other customers. (RPT-20995)
  • Typing an agent's name in the Contact History report search field returned no results. (RPT-20833)

Cumulative Update 2

Published July 5, 2022*

ACD

  • Rarely, refused contacts would become stuck in queue. (ORC-16698)
  • The Verified for Emergency Call Dispatching field appeared for all ACD user profiles, even for those in countries where emergency calling isn't supported yet. (AUI-15826)
  • When an employee was deactivated, the Edit button in ACD Users disappeared from the General tab. (AA-9601)

API

  • GET /points-of-contacts returned isActive and outboundSkill with values of 0 and 1 instead of true and false. (API-9378)

Developer Portal

  • The GET /points-of-contact response included totalRecords, which didn't appear in the model in the Developer Portal. (API-9377)

Platform

  • Some users couldn't log in with SSO using Chrome. (UH-21319)

CXone Recording

  • Due to an issue on the Media server, recorded calls appeared with the wrong audio. (MS-5645)

Reporting & Dashboards

  • The Agent Timecard report showed incorrect agent login times. (RPT-19852)

Studio

  • Clicking Save As in a phone script that contained a ReqAgent action returned an error. (STX-15100)

Cumulative Update 1

Published June 22, 2022*

ACD

  • Users could edit an existing ACD skill, clear the Service Level settings, and save it. (AUI-15485)
  • When ACD User custom fields included more than 50 characters, the field names were truncated. (ORC-16695)
  • In some environments, opening a user profile in the ACD Users page returned an error. (AUI-15304)

API

  • When an API call included a skip parameter, the response didn't reflect it. (API-9101)

PartnerHub

  • In some environments, Partner Log Reader requests sometimes became stuck in the In Process state. (PS-12016)

Personal Connection

  • In Skill Control, the Abandon % sometimes showed a negative value. (OB-10052)

Reporting & Dashboards

  • The SMS Transcripts data download report (555) and the Contact Details report didn't always show SMS transcripts. (RPT-9780)

  • For some ACD skill metrics, estimated values were not being cleared completely. This sometimes caused undesirable results. (ORC-16632)

Summer 2022 Release

Published June 7, 2022*

ACD

  • Changes logged in the Audit History tab were sometimes attributed to the wrong user. (AUI-14677)  
  • The business unit Audit History tab didn't show events for the Add All and Remove All actions for child business units. (AUI-14210)

Admin

  • In some environments, the team Contact Settings tab wasn't updating. (API-9015)
  • Sometimes new employee accounts didn't receive account verification emails. (CXAPP-17282)
  • ACD permissions showed two Hide Agent Identity Information permissions, and only one of them worked. (UH-20374)

Agent for Oracle Service Cloud

  • When an agent received an anonymous call, Agent for Oracle Service Cloud returned a memory error. The agent state became out of sync and the agent couldn't log out. (OSC-3863)

API

  • Some environments experienced intermittent connectivity issues and unexpected responses from CXone through API requests. (PAPI-4247)

  • The completed contacts response contained an ACWSeconds property that wasn't in camel case. (API-8957)

Cloud Storage Services

  • Retrieval jobs for long-term storage would sometimes get stuck in a WaitingUnArchive state. (STO-5986)
  • In some geographic regions, files didn't always successfully move to long-term storage. (STO-5916)

Developer Portal

  • The /contacts/completed documentation didn't specify that the tags parameter can only filter off the TagId. (IDE-806)

  • Under certain conditions, using the "try it out" feature returned an undocumented error. (API-8194)

Digital First Omnichannel

  • In some environments, chat windows were blank for the agent. (DE-31642)
  • Under certain conditions, custom fields weren't resetting between a contact's live chat cases. (DE-30799)
  • In some environments, outbound SMS messages weren't being billed properly. (DE-30361)
  • If a contact replied to a case before the agent had closed it, the agent couldn't see the message in the case until they refreshed. (DE-29923)
  • Customer cards sometimes didn't appear on YouTube cases. (DE-29186)
  • Filtering reports by agent didn't return results unless the filter text contained at least three characters. (DE-28698)
  • Some environments didn't allow creating a direct message channel for Twitter. (DE-28601)
  • Creating a CXone email channel with a special character in the From Address field resulted in the channel not working. No error was displayed. (DE-28429)
  • Some attachments, including .PDF and .DOCX files, couldn't be sent from an agent to a contact. (DE-28312)
  • Giving the "Reply" permission on a channel to a user role produced the error "There was something wrong. Try again or please contact administrator." (DE-27365)
  • Agents couldn't reply to a thread using a different channel unless all channels had reply approval. (DE-27300)
  • For certain configurations, sending a case as resolved would cause the application to crash. (DE-24371)
  • Generating DFO reports would return in the error message "An error has occurred. Please try again." (DE-24194)
  • Post details took a long time to load when an email thread was long. (DE-23568)
  • Rarely, cases would get stuck and not route to available agents. (DE-23072)

PartnerHub

  • Modifying a port limit sometimes resulted in a user not authorized error. (PS-10308)
  • In environments with an MS Teams subscription and APIs, sometimes the app wouldn't load due to high CPU utilization. (PIN-3899)
  • In some environments, activating Presence Sync resulted in an invalid BU ID error. (PIN-3893)
  • In some environments, Presence Sync was not updating for some directory contacts. (PIN-3835)
  • Dialing patterns didn't always apply to all numbers in a directory. (PIN-3831)
  • Presence Sync with MS Teams didn't always update the agent state in MAX or on the Contacts page. (PIN-3826)
  • Not all users appeared in the list when you assigned agents in Directory Sync. (PIN-3825)
  • When you assigned a default team in Directory Sync, users could see the directory in MAX but they couldn't see any of the default team members in the assigned users list. (PIN-3825)
  • Environments in the Australia region couldn't upgrade directories to Dir 2.0. (PIN-3818)
  • Directory Sync sometimes wouldn't update after changes were made in MS Teams. (PIN-3792)
  • Presence Sync for MS Teams sometimes didn't correctly automap users. (PIN-3723)
  • Some environments couldn't start the Presence Sync account mapping process. (PIN-3565)

Platform

  • Some organizations temporarily lost access to their CXone system. (TM-5318)
  • From the ACD and WFI apps, the navigation menu showed incorrect and missing icons. (AUI-13687)

CXone Recording

  • The time of the call in the voice file name was defined in 12-hour format instead of 24-hour format. (CXREC-19211)

Reporting & Dashboards

  • Some environments lost access to Dashboard following a permission migration. (SC-23860)
  • Logos configured in a custom report header sometimes didn't appear in XLSX reports. (RPT-19998)
  • Rarely, a user couldn't navigate to the Contact History report. It would redirect indefinitely. (API-9093)

Security & Compliance

  • The Reset Password email wasn't always sent. (SC-23030)
  • Users were unable to navigate to ACD, Reporting, or Dashboards in a FedRAMP SAML2 SSO environment. (SC-24600)

Virtual Agent Hub

  • In some environments, call scripts switched from Google WaveNet to CXone voice for text-to-speech. (AAI-2750)   
  • Some environments experienced utterance timeout errors and null audio transmissions. (AAI-2569)
  • Some environments experienced slow Google Dialogflow CX response times. (AAI-2490)

WFM

  • When filtered by the scheduling unit. Manage request was unable to show the person who reviewed the request.(CXWFM-26316)
